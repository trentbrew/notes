# Polycal Overview (working title)

## Overview

Polycal is a calendar application for multiple users or organizations to manage events.

## Features

- Integration with EventBrite
- Integration with Google Calendar
- Integration with Outlook
- Webflow plugin
- Email notifications
- Zapier Support

## Stack

Frontend

- Nuxt3
- Vue3
- Pinia
- Tailwind CSS
- DaisyUI

Backend

- Pocketbase
- Express
- Vercel

Project Management

- GitHub
- Notion
- GSuite
